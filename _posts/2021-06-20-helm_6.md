## Modify values.yaml and configmap.yaml

values.yaml  
```yaml
costCode: TEST
projectCode: aazzyy
infra:
  zone: a,b,c
  region: us-e
tags:
  machine: frontdrive
  rack: 4c
  drive: ssd
  vcard: 8g
LangUsed:
  - Python
  - Ruby
  - Java
  - Scala
```

configmap.yaml  
{% raw  %}
	apiVersion: v1
	kind: ConfigMap
	metadata:
	  name: {{ .Release.Name }}-configmap
	  labels:
	  {{- with .Values.tags }}
	  first: {{ .machine }}
	  second: {{ .rack }}
	  third: {{ .drive }}
	data:
	  myvalue: "Sample Config Map"
	  costCode: {{ .Values.costCode }}
	  Zone: {{ quote .Values.infra.zone }}
	  Region: {{ quote .Values.infra.region }}
	  ProjectCode: {{ upper .Values.projectCode }}
	  Lang Used: |-
	   {{- range .Values.LangUsed }}
	   - {{ . | title | quote }}
	   {{- end }}
{% endraw %}
