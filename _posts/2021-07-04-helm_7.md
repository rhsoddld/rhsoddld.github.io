## Modify configmap.yaml

values.yaml  
```yaml
costCode: TEST
projectCode: aazzyy
infra:
  zone: a,b,c
  region: us-e
tags:
  machine: frontdrive
  rack: 4c
  drive: ssd
  vcard: 8g
LangUsed:
  - Python
  - Ruby
  - Java
  - Scala
```

#### first example  

configmap.yaml  
{% raw  %}
	apiVersion: v1
	kind: ConfigMap
	metadata:
	  name: {{ .Release.Name }}-configmap
	  labels:
	  {{- with .Values.tags }}
	  first: {{ .machine }}
	  second: {{ .rack }}
	  third: {{ .drive }}
	  {{- end }}
	data:
	  myvalue: "Sample Config Map"
	  costCode: {{ .Values.costCode }}
	  Zone: {{ quote .Values.infra.zone }}
	  Region: {{ quote .Values.infra.region }}
	  ProjectCode: {{ upper .Values.projectCode }}
	  {{- $relname := .Release.Name }}
	  {{- with .Values.tags }}
	  Machine Type: {{ .machine | default "NA" | quote }}
	  Rack ID: {{ .rack | quote }}
	  Storage Type: {{ .drive | upper | quote }}
	  Video Card: {{ .vcard | quote }}
	  Release: {{ $relname }}
	  {{- end }}
{% endraw %}

	    {{- range $index, $topping := .Values.LangUsed }}
	    - {{ $index }} : {{ $topping }}
	    {{- end }}
